## ✨ Overview

`@fluixi/icon-tabler` is the **Tabler Icons** data package for the Fluixi icon system. It ships the raw SVG/icon data that powers the ergonomic umbrella packages. Source artwork: [Tabler Icons](https://tabler.io/icons) (licensed **MIT**).

Variants: `filled` · `outline`

> **Most users want the umbrella packages instead**, which provide tree-shakeable named exports, sprites, and CDN bundles:
>
> - **SVG strings / sprites / CDN** → [`@fluixi/icons`](../../icons-api) — `import { TrHome } from '@fluixi/icons/tabler/filled'`
> - **JSX components** → [`@fluixi/icons-jsx`](../../icons-jsx) — `import { TrHome } from '@fluixi/icons-jsx/tabler/filled'`

## 📦 Installation

```ini
# .npmrc

//npm.pkg.github.com/:_authToken=${NODE_AUTH_TOKEN}
```

```sh
pnpm add @fluixi/icon-tabler
```
